# Источники задач и контракты

Программа принимает задачи из различных источников и сохраняет их.

## Интерактивное взаимодействие

Вы можете взаимодействовать с программой с помощью 5 режимов:

1. Загрузить задачи из файла
2. Сгенерировать задачи(затем вы введёте число задач)
3. Загрузить задачи из API
4. Показать все задачи
5. Выход из программы

## Установка зависимостей и запуск

`make build`	    Первоначальная настройка проекта (установка uv, создание venv, установка зависимостей)
`make lint` 	    Проверка качества кода (mypy, ruff форматирование и линтинг)
`make start_test`	Запуск тестов и проверка покрытия кода
`make start`	    Запуск программы

Для полной установки зависимостей введите `make build`, затем `source .venv/bin/activate`.
